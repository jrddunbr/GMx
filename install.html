Install

This page details how to get a modded minecraft client running.
One can also use MultiMC - install it, then create a new 1.7.10 instance, then once it's created, click Install Forge, and then click Loader Mods, add, and then select the mods. Don't forget to setup an account in the upper right as well.

Contact Jared D. for more details (see main page).

Windows

  1. Install Java 8 (and not Java 9)
  2. Download the multiplatform Java .jar file from the Minecraft site
  3. Launch the jarfile and log into the Minecraft Launcher
  4. Close the Launcher and fetch the Forge Installer
  5. Launch the Forge Installer, and install the client to the default directory.
  6. Launch the Minecraft Launcher .jar file and select the Forge profile and launch the game
  7. Close Minecraft
  8. Download the mods zip archive
  9. Extract the mods to the C:\Users\<name>\AppData\Roaming\.minecraft\mods folder, and create it if it doesn’t exist
  10. Run the Minecraft Launcher, launch the Forge profile, profit

Linux

  1. Install Java 8 for your distro. You should really use Java 8. Java 7 or 9 are not guranteed to work well
  2. Download the .jar from Minecraft’s site
  3. Open a terminal in the folder where the jarfile is, and run java -jar Minecraft.jar, and log into the launcher
  4. Close the Launcher and fetch the Forge Installer
  5. Run that: java -jar forge-installer.jar
  6. Launch the Minecraft Launcher and select the Forge profile, and launch the game
  7. Close Minecraft
  8. Download the mods zip archive
  9. Extract the mods to the ~/.minecraft/mods folder and create it if it doesn’t exist
  10. Run the Minecraft Launcher, launch the Forge profile, profit
