Install

This page details how to get a modded minecraft client running.

One can use MultiMC, it's very easy to use, and cross platform.

The other option is to use the Mojang Launcher shipped with Minecraft by default, but be warned that sometimes it breaks.

Using the MultiMC Launcher

  1. Install Java for your platform.
    • Windows: Install Java 8 (not Java 9 or Java 10!)
    • Linux: Install Java 8 for your distro. (not Java 9 or Java 10)
      • Ubuntu and Debian:sudo apt install openjdk-8-jre
      Arch Linux and Manjaro:sudo pacman -Syu jre8-openjdk
      Fedora, CentOS, and RHEL(?!):sudo yum install java-1.8.0-openjdk
  2. Download MultiMC for your platform.
    • AUR Users: Install multimc5 from the AUR.
  3. Launch MultiMC
  4. Using the account manager in the upper right, sign into your Mojang account
  5. Right click in the large pane, and select "Create Instance". If you are using the 007 pack, you can also just use the Import function, and you're done.
  6. Click the 3 dots to the right of the version selector, under Vanilla Minecraft (don't let this fool you), and select the correct version for the pack. Click OK.
  7. Right click on the icon in the pane, and select "Edit Instance"
  8. Click the Install Forge button. Select the most recent version. Particularly, for 1.7.10, the version with a bug next to it is completely stable.
  9. On the left menu, click "Loader Mods"
  10. On the right menu, click "Add", then add the .jar files - be sure to EXTRACT your zip file the mods came in completely to a folder. It will NOT work from inside the .zip
  11. Click the "Launch" button at the bottom right. In the future, you can just launch the instance by double clicking on it, profit

Using the Mojang Launcher

Windows

  1. Install Java 8 (not Java 9 or Java 10)
  2. Download the multiplatform Java .jar file from the Minecraft site (there is also the jank .exe which I don't recommend)
  3. Launch the jarfile and log into the Minecraft Launcher
  4. Close the Launcher and fetch the Forge Installer
  5. Launch the Forge Installer, and install the client to the default directory.
  6. Launch the Minecraft Launcher .jar file and select the Forge profile and launch the game
  7. Close Minecraft
  8. Download the mods zip archive
  9. Extract the mods to the C:\Users\<name>\AppData\Roaming\.minecraft\mods folder, and create it if it doesn’t exist
  10. Run the Minecraft Launcher, launch the Forge profile, profit

Linux

  1. Install Java 8 for your distro (not Java 9 or Java 10).
  2. Download the .jar from Minecraft’s site
  3. Open a terminal in the folder where the jarfile is, and run java -jar Minecraft.jar, and log into the launcher
  4. Close the Launcher and fetch the Forge Installer
  5. Run that: java -jar forge-installer.jar
  6. Launch the Minecraft Launcher and select the Forge profile, and launch the game
  7. Close Minecraft
  8. Download the mods zip archive
  9. Extract the mods to the ~/.minecraft/mods folder and create it if it doesn’t exist
  10. Run the Minecraft Launcher, launch the Forge profile, profit
